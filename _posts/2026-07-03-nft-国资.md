## 国资问题

国资的作用就是按照中央党国务院和省政府的计划和需求设计项目的机会。有了项目再和各个政府商量如何符合项目资本的需求。最终要负责项目成功。国资会参与多轮的项目，也会连续和各个政府和伙伴合作。

每一次项目成功，下一轮更顺利了。项目和五年计划吻合无缝对接。国资信任为重要。

我们以下举个例子，分析相关刚出的15五年计划。也可以多加雷斯细节。

### 例如项目：内蒙古AI算力城

按照15五年计划建立一个国际级算力中心（10亿瓦级）。

<!-- 照片来自 https://lcj.yn.gov.cn/html/2022/guonei_0812/66779.html -->

![建立计算城](/assets/images/2026-07-03-nft-国资.webp)

我们先分析在内蒙古该项目的需求：

- 🟢 地面
- 🟢 矿场的煤炭
- 🟢 发电厂（10亿瓦级）
- 🟢 算力城外壳设计
- 🟢 算力城外壳建立（包电力、电池、散热，外网）
- 🟢 算力城内网络（服务器与服务器）
- 🟠 了解应用需求
- 🟠 服务器
- 🟠 AI芯片

在国外类似资本需求大约500亿美元（房地产、建设、发电厂、服务器、芯片）。在国内估计会便宜一些。但是实现该项目会需要国资和省、市政府和其他方大力支持。

:information_source: 关心点：以上的资本需求大大大量在于**🟠服务器**和**🟠AI芯片**。

## 国资和私资对比

我已经分析过如何在蒙古建立算力中心。外蒙私立、内蒙私立和内蒙国资项目都不同。以下对比项目**基础**：

|                                              | 外蒙私立                  | 内蒙私立                  | 内蒙国资                  |
| -------------------------------------------- | ------------------------- | ------------------------- | ------------------------- |
| 🟢 地面                                       | :x: 不能保证              | :x: 不能保证              | :white_check_mark: 可保证 |
| 🟢 矿场的煤炭                                 | :white_check_mark: 可以买 | :x: 不能保证              | :white_check_mark: 可保证 |
| 🟢 发电厂                                     | :x: 不能保证              | :x: 不能保证              | :white_check_mark: 可保证 |
| 🟢 算力城外壳设计                             | :white_check_mark: 可以做 | :x: 不能保证              | :white_check_mark: 可保证 |
| 🟢 算力城外壳建立（包电力、电池、散热，外网） | :white_check_mark: 可以做 | :x: 不能保证              | :white_check_mark: 可保证 |
| 🟢 算力城内网络                               | :white_check_mark: 可以做 | :white_check_mark: 可以做 | :white_check_mark: 可保证 |

剩下的算力部分更难说：

|                | 外蒙私立                    | 内蒙私立                  | 内蒙国资                    |
| -------------- | --------------------------- | ------------------------- | --------------------------- |
| 🟠 了解应用需求 | :white_check_mark: 懂得多   | :white_check_mark: 懂得多 | :grey_question: 可问企业    |
| 🟠 服务器       | :white_check_mark: 可以买   | :white_check_mark: 可以买 | :white_check_mark: 用国内的 |
| 🟠 AI芯片       | :white_check_mark: 用国外的 | :grey_question:           | :white_check_mark: 用国内的 |

项目的**基础**国资十分可以做得到。**私立资本无法保证项目实现。**

国资自己买**服务器**和**AI芯片**有可能买错款。还有这一部分是大大大量资本需求。

到这一点我们可以分析NFT/RWA资源如何帮助国资实现项目难点，还有西方的NFT/RWA和中国内应怎么不一样。

## 如何用NFT/RWA资源帮助项目

目前的困难是：

- 最终算力城要符合企业算力需求。
- 资本参与方需要确信项目能够顺利实现。
- **服务器**和**AI芯片**需求可能比国资提出过类似的项目更大。

还有不要忘记国资强项：

- 只有国资能保证各个省市政府的合作。
- 国资获得面积、发电证和别的证都不紧。
- AI大算力是15五年计划的目标企业。
- 国资能带来N多企业方介绍新的企业方式。

RWA资源可以带来一个合适的资本方式。可能更适合该项目。我们先对比几种资本方式：

| 投资           | 贷款/债券      | ⭐️ 质押             |
| -------------- | -------------- | ------------------ |
| 项目成功后分红 | 项目还在发利息 | 项目实现后有权合作 |

我们可以想象算力城是一个多横多竖的格子。

| 1号位置      | 2号位置      | 3号位置 | 4号位置 | 5号位置 |
| ------------ | ------------ | ------- | ------- | ------- |
| 。。。。。。 | 。。。。。。 |         |         |         |
|              |              |         |         |         |
|              |              |         |         |         |
|              |              |         |         |         |

每一个格子有一定的面积、一定的电力容量、一定的内外网、一定的散热限制。

如果国资已经通过所有的面积、发电、类似的证那就可以拍卖质押券。项目建立好之后谁持有该券，就有权利入驻。

**:star: 质押方提供自己的服务器和 AI 芯片。质押方入驻后仍需支付房租和电费。**

这些质押券可按市场价格拍卖给企业方。

这种资本方式可以带来大量早期的资本。也会减少国资的风险。

**:star: 通过质押资本，参与方为什么最相信项目会实现？因为政府方实现之后政府方还可以赚很多房租和电费。**

通过 NFT/RWA 技术，质押方持有的权益仍有流动性。在合规的国产区块链上（如果政府允许），A 方可以在二手市场持有的位置。A 方和 B 方也可以交换位置（AI 服务器对距离和位置非常敏感）。

西方的NFT/RWA和中国内应技术怎么不一样？

| 西方RWA资源       | 中国内资源       |
| ----------------- | ---------------- |
| NFT可以金融化     | 政府可以禁止     |
| 匿名方可以获得NFT | 过链可禁止       |
| 可用去中心化金融  | 可绑定人民币交易 |

除了技术意外，私立公司更喜欢高风险项目。他们不依赖政府贷款和质押模式。他们会到各个国家参与证券交易。国资的风险偏好不同，因此质押方式可能更适合。
